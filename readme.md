Этот проект - это простой пример использования библиотеки tinyusb для создания виртуального USB-устройства на ESP32. В данном случае, это виртуальный диск (RAM Disk), который может быть использован для хранения данных.

Проект использует библиотеку tinyusb для управления USB-интерфейсом ESP32 и библиотеку mscusb для реализации протокола Mass Storage (MS).

Программа создает виртуальный диск, который может быть подключен к компьютеру через USB и использован для хранения файлов. Диск хранит данные в оперативной памяти ESP32.

Этот проект может быть полезен для тестирования и разработки устройств с USB-интерфейсом, таких как флешки или внешние жесткие диски.
